## Installation

```bash
# npm
npm i @morfeo/react

# yarn
yarn add @morfeo/react
```

**@morfeo/react** re-expose [@morfeo/hooks](./hooks.mdx) and [@morfeo/web](./web.mdx), we suggest you to read their documentations.
