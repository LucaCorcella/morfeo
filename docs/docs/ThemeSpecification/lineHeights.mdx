## Interface

### Structure

```typescript
interface LineHeights {
  [key: string]: string;
}

type LineHeight = keyof LineHeights;
```

### Usage

```typescript
// Types are re-exported for ease of use by other packages
// like @morfeo/core, @morfeo/web or @morfeo/react
import { LineHeight, LineHeights } from '@morfeo/spec';
```

## Examples

Using the [CLI compose command](../Features/CLI/compose.mdx):

```typescript title="lineHeights.morfeo.ts"
// Remove this line to add the slice to all of your themes
export const themeName = 'My Theme Name';
// Optional if you name the file lineHeights.morfeo.ts
export const sliceName = 'lineHeights';

export default {
  body: '1.3em',
  heading: '1.6em',
};
```

Or manually:

```typescript title="lineHeights.ts"
import { LineHeights } from '@morfeo/spec';

const lineHeights: LineHeights = {
  body: '1.3em',
  heading: '1.6em',
};
```

```typescript title="theme.ts"
import { lineHeights } from './lineHeights';

const myTheme = {
  lineHeights,
  ...restOfTheTheme,
};

morfeo.setTheme('My Theme Name', myTheme);
```

## Extend Slice with custom properties

If you're using typescript and you add custom colors to your theme configuration, you'll need to merge
the default `LineHeights` interface with your custom one.
We suggest creating a declaration file (for example: `morfeo.d.ts` or `types.d.ts`) like the following one:

```typescript
import { lineHeights } from './path/to/your/custom/lineHeights';

type MyLineHeights = typeof lineHeights;

declare module '@morfeo/spec' {
  export interface LineHeights extends MyLineHeights {}
}
```

Read more about how to extend the default Morfeo Theme [here](../Features/extendible.mdx).
