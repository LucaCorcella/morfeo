## Interface

### Structure

```typescript
interface BorderWidths {
  [key: string]: string;
}

type BorderWidth = keyof BorderWidths;
```

### Usage

```typescript
// Types are re-exported for ease of use by other packages
// like @morfeo/core, @morfeo/web or @morfeo/react
import { BorderWidth, BorderWidths } from '@morfeo/spec';
```

## Examples

Using the [CLI compose command](../Features/CLI/compose.mdx):

```typescript title="borderWidths.morfeo.ts"
// Remove this line to add the slice to all of your themes
export const themeName = 'My Theme Name';
// Optional if you name the file borderWidths.morfeo.ts
export const sliceName = 'borderWidths';

export default {
  none: '0px',
  '2xs': '0.2px',
  xs: '0.5px',
  s: '1px',
  m: '2px',
  l: '4px',
  xl: '6px',
  '2xl': '8px',
};
```

Or manually:

```typescript title="borderWidths.ts"
import { BorderWidths } from '@morfeo/spec';

const borderWidths: BorderWidths = {
  none: '0px',
  '2xs': '0.2px',
  xs: '0.5px',
  s: '1px',
  m: '2px',
  l: '4px',
  xl: '6px',
  '2xl': '8px',
};
```

```typescript title="theme.ts"
import { borderWidths } from './borderWidths';

const myTheme = {
  borderWidths,
  ...restOfTheTheme,
};

morfeo.setTheme('My Theme Name', myTheme);
```

## Extend Slice with custom properties

If you're using typescript and you add custom colors to your theme configuration, you'll need to merge
the default `BorderWidths` interface with your custom one.
We suggest creating a declaration file (for example: `morfeo.d.ts` or `types.d.ts`) like the following one:

```typescript
import { borderWidths } from './path/to/your/custom/borderWidths';

type MyBorderWidths = typeof borderWidths;

declare module '@morfeo/spec' {
  export interface BorderWidths extends MyBorderWidths {}
}
```

Read more about how to extend the default Morfeo Theme [here](../Features/extendible.mdx).
