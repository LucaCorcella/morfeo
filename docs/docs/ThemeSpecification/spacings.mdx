## Interface

### Structure

```typescript
interface Spacings {
  [key: string]: string;
}

type Spacing = keyof Spacings;
```

### Usage

```typescript
// Types are re-exported for ease of use by other packages
// like @morfeo/core, @morfeo/web or @morfeo/react
import { Spacing, Spacings } from '@morfeo/spec';
```

## Examples

Using the [CLI compose command](../Features/CLI/compose.mdx):

```typescript title="spacings.morfeo.ts"
// Remove this line to add the slice to all of your themes
export const themeName = 'My Theme Name';
// Optional if you name the file spacings.morfeo.ts
export const sliceName = 'spacings';

export default {
  xxs: '8px',
  xs: '16px',
  s: '24px',
  m: '32px',
  l: '40px',
  xl: '48px',
  xxl: '56px',
  none: '0px',
};
```

Or manually:

```typescript title="spacings.ts"
import { Spacings } from '@morfeo/spec';

const spacings: Spacings = {
  xxs: '8px',
  xs: '16px',
  s: '24px',
  m: '32px',
  l: '40px',
  xl: '48px',
  xxl: '56px',
  none: '0px',
};
```

```typescript title="theme.ts"
import { spacings } from './spacings';

const myTheme = {
  spacings,
  ...restOfTheTheme,
};

morfeo.setTheme('My Theme Name', myTheme);
```

## Extend Slice with custom properties

If you're using typescript and you add custom colors to your theme configuration, you'll need to merge
the default `Spacings` interface with your custom one.
We suggest creating a declaration file (for example: `morfeo.d.ts` or `types.d.ts`) like the following one:

```typescript
import { spacings } from './path/to/your/custom/spacings';

type MySpacings = typeof spacings;

declare module '@morfeo/spec' {
  export interface Spacings extends MySpacings {}
}
```

Read more about how to extend the default Morfeo Theme [here](../Features/extendible.mdx).
