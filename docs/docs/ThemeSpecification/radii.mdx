## Interface

### Structure

```typescript
interface Radii {
  [key: string]: string;
}

type Radius = keyof Radii;
```

### Usage

```typescript
// Types are re-exported for ease of use by other packages
// like @morfeo/core, @morfeo/web or @morfeo/react
import { Radius, Radii } from '@morfeo/spec';
```

## Examples

Using the [CLI compose command](../Features/CLI/compose.mdx):

```typescript title="radii.morfeo.ts"
// Remove this line to add the slice to all of your themes
export const themeName = 'My Theme Name';
// Optional if you name the file radii.morfeo.ts
export const sliceName = 'radii';

export default {
  xxs: '4px',
  xs: '8px',
  s: '12px',
  m: '16px',
  l: '20px',
  xl: '24px',
  xxl: '28px',
  none: '0px',
  round: '50%',
};
```

Or manually:

```typescript title="radii.ts"
import { Radii } from '@morfeo/spec';

const radii: Radii = {
  xxs: '4px',
  xs: '8px',
  s: '12px',
  m: '16px',
  l: '20px',
  xl: '24px',
  xxl: '28px',
  none: '0px',
  round: '50%',
};
```

```typescript title="theme.ts"
import { radii } from './radii';

const myTheme = {
  radii,
  ...restOfTheTheme,
};

morfeo.setTheme('My Theme Name', myTheme);
```

## Extend Slice with custom properties

If you're using typescript and you add custom colors to your theme configuration, you'll need to merge
the default `Radii` interface with your custom one.
We suggest creating a declaration file (for example: `morfeo.d.ts` or `types.d.ts`) like the following one:

```typescript
import { radii } from './path/to/your/custom/radii';

type MyRadii = typeof radii;

declare module '@morfeo/spec' {
  export interface Radii extends MyRadii {}
}
```

Read more about how to extend the default Morfeo Theme [here](../Features/extendible.mdx).
